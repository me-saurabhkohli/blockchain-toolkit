Installation

1

Install from VS Code Marketplace

The easiest way to install the Blockchain Toolkit extension:

VS Code Command 
ext install blockchain-toolkit

Or search for "Blockchain Toolkit" in the Extensions panel (Ctrl+Shift+X)

2

Verify Installation

After installation, you should see:

  • Blockchain Toolkit icon in the Activity Bar
  • New commands in the Command Palette (Ctrl+Shift+P)
  • "Blockchain Toolkit" output channel
3

Prerequisites Check

Ensure you have the following installed:

Node.js

Version 16 or higher

node --version

Package Manager

npm, yarn, or pnpm

npm --version

Git

For version control

git --version

First-Time Setup

Auto-Configuration

The extension automatically detects and configures itself based on your project structure. No manual setup required for most projects!

Automatic Detection

The extension automatically scans your workspace for:

  • Hardhat configuration files
  • Truffle configuration files
  • Foundry project structure
  • Solana/Anchor projects
  • Package.json dependencies

Configuration Options

Customize the extension behavior:

  • Default compiler versions
  • Gas reporting settings
  • Debug mode preferences
  • Network configurations
  • Security analysis rules

Configuration Settings

Access settings via File → Preferences → Settings, then search for "blockchain":

settings.json 
{
  "blockchain.defaultNetwork": "localhost",
  "blockchain.enableGasReporting": true,
  "blockchain.autoCompile": false,
  "blockchain.solcVersion": "0.8.19",
  "blockchain.enableDebugMode": false,
  "blockchain.evmDebug": {
    "autoRefresh": true,
    "maxStackItems": 50,
    "enableGasAnalysis": true
  }
}

Framework Detection

The extension supports multiple blockchain frameworks and automatically detects which ones are present in your workspace.

Hardhat

Detects via:

  • hardhat.config.js
  • hardhat.config.ts
  • hardhat in dependencies

Truffle

Detects via:

  • truffle-config.js
  • truffle.js
  • truffle in dependencies

Foundry

Detects via:

  • foundry.toml
  • lib/ directory
  • forge/cast binaries

Solana/Anchor

Detects via:

  • Anchor.toml
  • Cargo.toml
  • programs/ directory

Multiple Frameworks

The extension can handle projects with multiple frameworks. Use Ctrl+Shift+P → Blockchain: Select Framework to switch between detected frameworks.

Creating Your First Project

Create a Hardhat Project

1

Initialize Project

mkdir my-hardhat-project
cd my-hardhat-project
npx hardhat init
2

Open in VS Code

code .

The extension will automatically detect the Hardhat configuration and activate.

3

Verify Setup

Check that you see:

  • "Hardhat" in the status bar
  • Blockchain Toolkit panel in the sidebar
  • Hardhat commands in Command Palette

Create a Truffle Project

1

Initialize Project

mkdir my-truffle-project
cd my-truffle-project
npx truffle init
2

Open in VS Code

code .

Create a Foundry Project

1

Install Foundry

curl -L https://foundry.paradigm.xyz | bash
foundryup
2

Initialize Project

forge init my-foundry-project
cd my-foundry-project

Create a Solana/Anchor Project

1

Install Anchor

cargo install --git https://github.com/coral-xyz/anchor avm --locked --force
avm install latest
avm use latest
2

Initialize Project

anchor init my-solana-project

Basic Development Workflow

1. Write Contracts

Create or edit your smart contracts in the contracts/ directory. The extension provides:

  • Solidity syntax highlighting
  • Code snippets
  • Auto-completion
  • Error highlighting

2. Compile Contracts

Use the command palette or click the compile button:

Ctrl+Shift+P → Blockchain: Compile

Or use the keyboard shortcut: Ctrl+Shift+B

3. Write Tests

Create test files in the test/ directory. The extension supports:

  • JavaScript/TypeScript tests
  • Solidity tests (Foundry)
  • Test debugging
  • Coverage analysis

4. Run Tests

Execute your test suite:

Ctrl+Shift+P → Blockchain: Test

View results in the integrated terminal and output panel.

5. Deploy Contracts

Deploy to local or remote networks:

Ctrl+Shift+P → Blockchain: Select Network
Ctrl+Shift+P → Blockchain: Deploy

Monitor deployment progress and results in real-time.
