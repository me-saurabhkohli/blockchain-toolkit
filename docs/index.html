Get Started in Minutes

Choose your preferred installation method

# Install from VS Code Marketplace
ext install blockchain-toolkit

# Or search "Blockchain Toolkit" in Extensions

Quick Start

# Open your blockchain project
code my-blockchain-project

# Use Command Palette
Ctrl+Shift+P ‚Üí Blockchain: Compile

# Or use sidebar navigation
# Install globally
npm install -g blockchain-toolkit-cli

# Or use with npx
npx blockchain-toolkit --help

Quick Start

# Analyze your project
blockchain-toolkit analyze ./my-project

# Compile contracts
blockchain-toolkit compile

# Deploy to network
blockchain-toolkit deploy --network goerli
